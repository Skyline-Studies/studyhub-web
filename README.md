# StudyHub Web
StudyHub Web is the web version of StudyHub, built with React and JavaScript/JSX.

## Features
- [Web](https://skyline-studies.github.io/studyhub-web/) and [mobile](https://github.com/Skyline-Studies/studyhub-mobile/releases) apps  
- Calendar with day, week, month, and year views  
- Add tasks to the calendar or to‑do list  
  - Calendar tasks save locally to your device/browser  
  - Listify tasks save to the cloud (requires login/signup)  
- Create notes that save locally (or use Listify for cloud‑saved notes)  
- AI studying (coming soon — includes a lock mode to discourage AI use during tests, plus optional usage logs for teachers, which is coming soon...)

## Installation
StudyHub Web runs in the browser — no downloads needed.  
Just visit: https://skyline-studies.github.io/studyhub-web/
