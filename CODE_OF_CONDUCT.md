# Code of Conduct

Welcome to StudyHub Web. To keep this project welcoming and enjoyable for everyone, we ask all contributors to follow these guidelines.

## Our Standards
Please:
- Be respectful and considerate  
- Use clear and constructive communication  
- Welcome beginners and new contributors  
- Focus on solving problems, not creating them  

We do not tolerate:
- Harassment or discrimination of any kind  
- Personal attacks or hostile behaviour  
- Spam, trolling, or intentionally disruptive actions  

## How to Report Issues
If you experience or witness behaviour that violates this Code of Conduct, please open an issue or contact a project maintainer privately. All reports will be handled respectfully.

## Enforcement
Maintainers may take appropriate action, including warnings or removal of contributions, if behaviour violates these guidelines.

## Scope
This Code of Conduct applies to:
- GitHub issues and pull requests  
- Discussions and comments  
- Any community spaces related to StudyHub Web  

Thank you for helping keep the Skyline Studies community positive and welcoming.
