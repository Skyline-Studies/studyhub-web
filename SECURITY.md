# Security Policy

## Reporting a Vulnerability
If you discover a security issue in StudyHub Web, please report it privately.  
Open a GitHub issue **without** including sensitive details, and a maintainer will contact you to continue the discussion securely.

## Supported Versions
StudyHub Web is actively developed.  
Only the latest version receives security updates or fixes.

## Guidelines
- Do not publicly disclose vulnerabilities before they are fixed  
- Provide clear steps to reproduce the issue (if possible)  
- Avoid testing in ways that could disrupt other users  
- Do not attempt to access data that isnâ€™t yours  

## Scope
This policy applies to:
- StudyHub Web  
- Any related frontend code in the Skyline Studies ecosystem  

Thank you for helping keep StudyHub Web safe for everyone.
